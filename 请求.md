# HTTP请求 (Fetch)

OTD Core 的 HTTP 请求模块基于 Alova 构建，提供统一的请求配置、拦截器处理和响应格式化功能。

## 概述

HTTP 请求模块具备以下核心能力：

- **基于 Alova**: 构建在现代化的 Alova 请求库之上
- **统一配置**: 提供开箱即用的应用级配置，减少重复代码
- **智能拦截器**: 自动处理认证、错误、Loading 状态等
- **类型安全**: 完整的 TypeScript 支持
- **灵活扩展**: 支持完全自定义配置的高级用法

## 快速开始

### 基础用法

```javascript
import { createAppAlova } from '@x-ui/otd-core';

// 创建应用级 API 实例
const api = createAppAlova({
  baseURL: 'https://api.example.com',
  token: 'your-auth-token',
  onSuccess: (data) => {
    console.log('请求成功:', data);
  },
  onError: (error) => {
    console.error('请求失败:', error.message);
  },
  onLoading: (loading) => {
    console.log('Loading状态:', loading);
  }
});

// 使用 Alova 的所有功能
const users = await api.Get('/users');
const user = await api.Post('/users', userData);
```

### Vue 3 组件中使用

```javascript
import { useRequest } from 'alova';

// 在组件中使用
export default {
  setup() {
    // 获取用户列表
    const getUserList = () => api.Get('/users');
    const { data: users, loading, error } = useRequest(getUserList());

    // 创建用户
    const createUser = (userData) => api.Post('/users', userData);
    const { send: sendCreateUser, loading: creating } = useRequest(createUser, {
      immediate: false
    });

    return {
      users,
      loading,
      error,
      sendCreateUser,
      creating
    };
  }
};
```

## 核心 API

### createAppAlova

创建应用级 Alova 实例的核心函数，提供统一的配置体验。

```typescript
function createAppAlova(config: AppAlovaConfig): AlovaInstance;

interface AppAlovaConfig {
  baseURL: string; // 基础 URL
  timeout?: number; // 请求超时时间，默认 10000ms
  token?: string; // 访问令牌，自动添加到 Authorization 头
  onSuccess?: (data: any) => void; // 成功响应回调
  onError?: (error: Error) => void; // 错误响应回调
  onLoading?: (loading: boolean) => void; // Loading 状态变化回调
  customHeaders?: Record<string, string>; // 自定义请求头
}
```

### 响应格式

统一的响应数据格式：

```typescript
interface BaseResponse<T = any> {
  code: number; // 状态码
  message: string; // 响应消息
  data: T; // 业务数据
  success: boolean; // 是否成功
}
```

### 错误处理

增强的错误类型和处理：

```typescript
enum ErrorType {
  NETWORK_ERROR = 'NETWORK_ERROR', // 网络错误
  HTTP_ERROR = 'HTTP_ERROR', // HTTP 错误
  BUSINESS_ERROR = 'BUSINESS_ERROR', // 业务错误
  TIMEOUT_ERROR = 'TIMEOUT_ERROR', // 超时错误
  CANCEL_ERROR = 'CANCEL_ERROR' // 取消请求
}

class FetchError extends Error {
  type: ErrorType;
  status?: number;
  response?: any;
}
```

## 拦截器机制

### 请求拦截器

自动处理常见的请求配置：

```javascript
// 自动添加的功能：
// 1. Authorization token
// 2. Content-Type: application/json
// 3. 自定义请求头
// 4. Loading 状态开始

const api = createAppAlova({
  baseURL: 'https://api.example.com',
  token: 'your-token', // 自动添加 Bearer token
  customHeaders: {
    // 自定义请求头
    'X-App-Version': '1.0.0'
  }
});
```

                    │                     │

## 使用示例

### 完整的应用配置

```javascript
// api/index.js
import { createAppAlova } from '@x-ui/otd-core';
import { ElMessage } from 'element-plus';

export const api = createAppAlova({
  baseURL: import.meta.env.VITE_API_BASE_URL,
  timeout: 15000,

  // 自动 token 管理
  token: localStorage.getItem('access_token'),

  // 统一成功处理
  onSuccess: (data) => {
    console.log('API 请求成功:', data);
  },

  // 统一错误处理
  onError: (error) => {
    if (error.type === 'HTTP_ERROR' && error.status === 401) {
      // 处理未授权
      localStorage.removeItem('access_token');
      router.push('/login');
    } else {
      ElMessage.error(error.message);
    }
  },

  // 全局 Loading 管理
  onLoading: (loading) => {
    store.commit('setGlobalLoading', loading);
  },

  // 自定义请求头
  customHeaders: {
    'X-App-Version': '1.0.0',
    'X-Client-Type': 'web'
  }
});
```

### RESTful API 封装

```javascript
// api/user.js
import { api } from './index.js';

// 用户相关 API
export const userAPI = {
  // 获取用户列表
  getUsers: (params) => api.Get('/users', { params }),

  // 获取用户详情
  getUser: (id) => api.Get(`/users/${id}`),

  // 创建用户
  createUser: (data) => api.Post('/users', data),

  // 更新用户
  updateUser: (id, data) => api.Put(`/users/${id}`, data),

  // 删除用户
  deleteUser: (id) => api.Delete(`/users/${id}`),

  // 上传头像
  uploadAvatar: (id, file) =>
    api.Post(`/users/${id}/avatar`, file, {
      headers: { 'Content-Type': 'multipart/form-data' }
    })
};
```

### 组件中使用

```vue
<template>
  <div>
    <div v-if="loading">加载中...</div>
    <div v-else-if="error">{{ error.message }}</div>
    <div v-else>
      <user-item v-for="user in users" :key="user.id" :user="user" @edit="editUser" @delete="deleteUser" />
    </div>

    <button @click="createUser" :disabled="creating">
      {{ creating ? '创建中...' : '创建用户' }}
    </button>
  </div>
</template>

<script setup>
import { useRequest } from 'alova';
import { userAPI } from '@/api/user';

// 获取用户列表
const { data: users, loading, error, send: refreshUsers } = useRequest(userAPI.getUsers);

// 创建用户
const { send: sendCreateUser, loading: creating } = useRequest(userAPI.createUser, {
  immediate: false
});

// 编辑用户
const editUser = async (user) => {
  try {
    await userAPI.updateUser(user.id, user);
    refreshUsers(); // 刷新列表
  } catch (error) {
    console.error('更新失败:', error);
  }
};

// 删除用户
const deleteUser = async (userId) => {
  try {
    await userAPI.deleteUser(userId);
    refreshUsers(); // 刷新列表
  } catch (error) {
    console.error('删除失败:', error);
  }
};

// 创建用户
const createUser = () => {
  sendCreateUser({
    name: '新用户',
    email: 'new@example.com'
  }).then(() => {
    refreshUsers(); // 刷新列表
  });
};
</script>
```

## HTTP 状态码处理

模块提供了完整的 HTTP 状态码枚举和错误处理：

```javascript
import { HttpStatus, ErrorType } from '@x-ui/otd-core';

// HTTP 状态码
HttpStatus.OK; // 200
HttpStatus.CREATED; // 201
HttpStatus.BAD_REQUEST; // 400
HttpStatus.UNAUTHORIZED; // 401
HttpStatus.FORBIDDEN; // 403
HttpStatus.NOT_FOUND; // 404
HttpStatus.INTERNAL_SERVER_ERROR; // 500

// 错误类型判断
if (error.type === ErrorType.HTTP_ERROR && error.status === HttpStatus.UNAUTHORIZED) {
  // 处理未授权错误
  redirectToLogin();
}
```

## Wormhole - Swagger 代码生成器 🕳️

Wormhole 是一个强大的 Swagger 文档自动化代码生成工具，可以根据 Swagger/OpenAPI 文档自动生成类型安全的 API 调用代码。

> **注意**: Wormhole 依赖 Node.js 环境，已独立分包为 `@x-ui/otd-core/fetch-wormhole`

### 核心特性

- ✅ **自动生成**: 从 Swagger 文档自动生成 API 函数和类型定义
- ✅ **智能合并**: 函数级别的增量更新，保留用户手动代码
- ✅ **TypeScript/JavaScript**: 同时支持 TS 和 JS
- ✅ **Mock 数据**: 自动生成 Mock 数据文件
- ✅ **多服务支持**: 按服务名自动分组

### 快速开始

```javascript
// wormhole.js
import { wormholeSwagger } from '@x-ui/otd-core/fetch-wormhole';

const config = {
  generator: [
    {
      swagger: {
        url: 'https://api.example.com/v3/api-docs/ALL',
        // 支持 Basic 认证
        auth: {
          username: 'admin',
          password: 'password'
        }
      },
      outputDir: './src/api',
      language: 'ts', // 'ts' | 'js'
      apis: ['/common/dict-item/list/enable', '/users'],
      mock: {
        enable: true,
        arrayLength: 5
      }
    }
  ]
};

// 执行生成
wormholeSwagger(config);
```

**运行脚本：**

```bash
node wormhole.js
```

### 智能合并功能

Wormhole 支持**函数级别的智能合并**，不会覆盖用户手动添加的代码：

```javascript
// 初次生成
export async function getUser(params) {
  /* 自动生成 */
}

// 用户手动添加
export function getUserWithCache(id) {
  const cached = localStorage.getItem(`user_${id}`);
  return cached ? JSON.parse(cached) : getUser({ id });
}

// 再次生成 → getUser 被更新，getUserWithCache 保留 ✅
```

**合并策略：**

- 🔄 **同名函数**: 替换为最新版本
- ➕ **新增函数**: 追加到文件末尾
- 📦 **用户代码**: 完全保留（包括自定义 imports、工具函数等）

### Mock 数据使用

配置中启用 `mock.enable: true` 后会生成 `mock.ts` 文件：

```javascript
const config = {
  generator: [{
    swagger: { url: 'https://api.example.com/swagger.json' },
    outputDir: './src/api',
    apis: ['/users'],
    mock: {
      enable: true,
      arrayLength: 5
    }
  }]
};
```

**使用方式：直接在生成的函数中添加 `useMock: true`**

```javascript
// 生成的代码
export async function getUser(params) {
  const url = '/users/:id';
  return api.Get(url, { params, useMock: true });  // 手动添加 useMock
}

export async function createUser(data) {
  const url = '/users';
  return api.Post(url, data, { useMock: true });  // 手动添加 useMock
}
```

> ⚠️ **注意**: 手动添加的 `useMock: true` 在重新生成时会被覆盖，建议仅在开发调试时使用。
