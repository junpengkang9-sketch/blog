# 微前端 (Micro)

OTD Core 的微前端模块提供了完整的微前端应用解决方案，支持多种微前端框架的自动检测和适配。

## 概述

微前端模块具备以下核心能力：

- **多框架支持**: 自动识别并适配 Wujie、Qiankun、Micro-app 等主流微前端框架
- **智能环境检测**: 自动检测运行环境，无需手动配置
- **统一应用创建**: 提供统一的 API 创建微前端应用，屏蔽框架差异
- **路由同步**: 支持父子应用路由状态同步
- **全局状态管理**: 提供应用实例的全局状态管理

## 快速开始

### 基础用法

```javascript
import { createMicroApp } from '@x-ui/otd-core';
import { createWebHistory } from 'vue-router';
import App from './App.vue';
import router from './router';

// 创建微前端应用
const microApp = await createMicroApp({
  name: 'package-app-name',
  app: App,
  router,
  setup(app) {
    // 配置应用插件、组件等
    app.use(SomePlugin);
  }
});
```

### 环境自适应

应用会自动检测运行环境并采用相应的处理策略：

```javascript
// 同一份代码在不同环境下自动适配
const microApp = await createMicroApp({
  name: 'package-app-name',
  app: App,
  router,
  autoMount: true // 独立环境自动挂载，微前端环境由父应用控制
});
```

## 支持的微前端框架

### Wujie 框架

```javascript
// Wujie 环境下的特殊处理
const microApp = await createMicroApp({
  name: 'package-app-name',
  app: App,
  router,
  setup(app) {
    // 应用配置
  }
});

// 框架会自动设置 window.__WUJIE_MOUNT 函数
// 支持父子应用路由同步
```

### Qiankun 框架

```javascript
// 配合 vite-plugin-qiankun 使用
import { qiankun } from '@x-ui/otd-core';

// vite.config.js 配置
export default {
  plugins: [
    qiankun('my-qiankun-app', {
      useDevMode: true
    })
  ]
};

// 应用入口
const microApp = await createMicroApp({
  name: 'package-app-name',
  app: App,
  router
});

// 自动设置 qiankun 生命周期钩子
```

### Micro-app 框架

```javascript
// Micro-app 环境自动适配
const microApp = await createMicroApp({
  name: 'package-app-name',
  app: App,
  router
});

// 自动处理 Micro-app 特有的环境变量和生命周期
```

## API 参考

### createMicroApp

创建微前端应用的核心函数。

```typescript
function createMicroApp(config: MicroAppConfig): Promise<MicroApp>;

interface MicroAppConfig {
  name: string; // 应用名称
  app: any; // Vue 根组件
  mode?: MicroAppMode; // 运行模式，通常自动检测
  setup?: (app: any) => void; // 应用配置回调
  requireAuth?: boolean; // 是否需要认证
  autoMount?: boolean; // 是否自动挂载，默认 true
  router?: any; // Vue Router 实例
}

interface MicroApp {
  readonly config: MicroAppConfig; // 应用配置
  readonly app: any | null; // Vue 应用实例
  mount(props?: any): Promise<void>; // 挂载应用
}
```

### 全局状态管理

```javascript
import { globalState, VueApp } from '@x-ui/otd-core';

// 获取当前应用实例
const currentApp = globalState.getCurrentApp();

// 获取当前 Vue 应用实例
const vueApp = globalState.getVueApp();

// 便捷访问器
const currentVueApp = VueApp.current;
```

### 环境检测函数

```javascript
import { detectEnvironment } from '@x-ui/otd-core';

// 检测当前运行环境
const envInfo = detectEnvironment();
console.log(`运行模式: ${envInfo.mode}`);
console.log(`父应用类型: ${envInfo.parentApp?.type}`);
```

## 注意事项

1. **应用名称**: 确保应用名称在微前端环境中唯一
2. **路由配置**: 微前端环境下路由可能需要特殊处理
3. **生命周期**: 不同微前端框架的生命周期处理方式不同
4. **全局变量**: 避免污染全局命名空间
5. **资源隔离**: 注意样式和脚本的隔离问题
